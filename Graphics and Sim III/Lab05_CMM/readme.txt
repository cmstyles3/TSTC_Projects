INSTRUCTIONS

CONTROLS
AT THE START SCREEN
Press [R] to reset highscore list
press [spacebar] to start game
press [escape] to exit the game

GAMEPLAY
press the arrow keys to move your polygon
press [S] to show the bounding volumes for the polygons.
press [P] to pause the game
press again to unpause
press [escape] to exit back to the start.


JOYSTICK CONTROLS 
(note this is for an xbox 360 controller)

AT THE START SCREEN
press the y button to reset highscores
press the start button to start game
and press the back button to exit game

GAMEPLAY
use the left stick to move your polygon
press the x button to show bounding volumes
press start to pause and unpause
press back to exit back to start.
